Elemento Samples

This folder contains different implementations of the TodoMVC application. Each sample uses Elemento and is based either on GWT or J2CL:

  • GWT: Pure Elemento based on GWT
  • J2CL: Pure Elemento based on J2CL
  • Crysknife: Elemento, Crysknife CDI, Crysknife templates based on J2CL

All samples use the same key to persist the todo items in the local storage. So you can switch between the samples and continue working on your tasks seamlessly ;-)
